# NodeSeek 增强助手

简洁高效的 NodeSeek 论坛增强油猴脚本。

## 功能特点

| 功能 | 说明 |
|------|------|
| **自动签到** | 每日自动签到，支持随机/固定鸡腿模式 |
| **最新留言预览** | 帖子标题旁显示最新评论（支持多页帖子） |
| **交易成交记录** | 侧边栏显示最近5笔已出/已收交易 |
| **抽奖帖汇总** | 侧边栏显示最新5个抽奖帖（区分进行中/已开奖） |
| **智能缓存** | 数据缓存5分钟，减少请求 |
| **深色模式** | 自动适配系统主题 |

## 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击安装脚本：[nodeseek-auto-checkin.user.js](./nodeseek-auto-checkin.user.js)
3. 访问 NodeSeek 论坛即可自动生效

## 配置选项

编辑脚本中的 `CONFIG` 对象：

```javascript
const CONFIG = {
    RANDOM_MODE: true,        // true: 随机鸡腿, false: 固定5鸡腿
    PREVIEW_MAX_LEN: 40,      // 评论预览最大字数
    SIDEBAR_COUNT: 5,         // 侧边栏显示数量
    CACHE_TTL: 5 * 60 * 1000  // 缓存时间(毫秒)
};
```
